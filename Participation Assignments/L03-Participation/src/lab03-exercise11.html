1. Describe the difference between a relative and absolute reference. When should each be used?

An absolute reference is used when referring to an external source and is used because you need to be very specific on where something is when working externally in order to make sure you are referencing the right thing. A relative reference is used when referring to something on the same server or in the same file structure and is less specific when pointing to whatever you are referring. This is because when working internally you don't need to worry as much about tracking things down in other locations.
